# Introduction to Shell Scripting

## What is Shell?
A shell is a program that allows users to interact with the operating system using commands.

## What is Shell Scripting?
Shell scripting is writing a sequence of commands in a file to automate tasks.

## What is Shebang?
Shebang (#!) tells the system which interpreter to use.

Example:
#!/bin/bash

## How to check shell type?
echo $0

## How to run a script?
chmod +x script.sh
./script.sh
